# Qualidade de instalação

Instalação correta deve considerar vácuo, suporte, dreno, distância, passagem da tubulação, ponto elétrico e teste final. Preço baixo demais pode cortar etapa importante, mas a resposta não deve atacar concorrente.

Quando o cliente comparar preço, explique o que precisa ser validado para evitar vazamento, ruído, perda de garantia e retrabalho.
