# Residencial comum

Atendimento residencial comum deve ser direto, educado e prático. O cliente normalmente quer instalação simples, higienização, manutenção de split, preço base ou agenda.

Para instalação de split, confirme cidade/bairro, BTUs, foto do local interno, foto do local externo, ponto elétrico exclusivo e distância aproximada. Se for acesso simples, pode informar a base de preço vigente. Se tiver telhado, altura, distância grande, dreno pendente ou elétrica fora do padrão, conduza para análise técnica.
