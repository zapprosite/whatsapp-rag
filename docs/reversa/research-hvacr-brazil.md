## 3. MELHORES PRÁTICAS: CHATBOT NO HVAC-R

### 3.1 Funcionalidades Críticas para Chatbot HVAC-R

| Categoria | Perguntas/Funções | Automação |
|-----------|------------------|-----------|
| **Emergências** | "Ar parou às 23h" | Coleta dados → dispatch |
| **Troubleshooting** | "AC fazendo barulho estranho" | Diagnóstico guiado passo-a-passo |
| **Agendamento** | "Preciso de visita amanhã" | Agenda slot disponível em tempo real |
| **Qualificação** | Tamanho casa, idade sistema, orçamento | Prepara técnico com contexto |
| **Rastreamento** | "Onde está o técnico?" | Status + instruções |
| **Lembretes** | Manutenção preventiva | Proativo outbound |

### 3.2 Métricas de Impacto

| Métrica | Valor |
|---------|-------|
| Conversões com chatbot (sem handoff) | ~70% |
| Aumento conversão com resposta <5s vs >5min | +391% |
| Satisfação cliente com chatbot 24/7 | +40% |
| Buscas emergências fora do horário comercial | 40% |
| Clientes que não deixam voicemail | 73% |

### 3.3 Por Que Chatbots Falham no HVAC-R

1. **Falta de dados de treinamento** — Chatbots genéricos sem conhecimento de refrigerantes, BTUs, tipos de sistema
2. **Não lidar com urgência** — Sistema não prioriza emergências térmicas
3. **Sem integração com field service** — Cliente repete informações ao técnico
4. **Ignorar canal mobile** — Não funcionar bem em WhatsApp
5. **Não qualificar orçamento** — Técnicos perdem tempo em clientes sem condição

## 4. ANÁLISE DE GAP — Onde a Refrimix Pode Liderar

### 4.1 Gaps de Mercado que Nenhuma Empresa Resolve Bem

| Gap | Por que ninguém resolve | Como a Refrimix resolve |
|-----|------------------------|------------------------|
| **Atendimento 24/7 com técnico qualificado** | Empresas têm chatbot mas não têm técnicos disponíveis 24h | Chatbot como primeiro contato, escalação contextual |
| **Orçamento transparente antes da visita** | Ninguém publica preços — esconde para "agendar orçamento" | Questionário → preço fixo ou faixa, sem jogo de esconde-esconde |
| **Manutenção preventiva proativa** | Todas focam em correção, não prevenção | Chatbot que lembre e agende manutenção antes da quebra |
| **Geolocalização + qualificação** | Atendimento é genérico, não지역 | Qualificação por tipo/size/local → routing para técnico certo |
| **Histórico do equipamento** | Cliente nunca sabe o histórico, técnico sempre pergunta de novo | LeadState com histórico → técnico já chega com contexto |

### 4.2 O Diferencial Refrimix

**Modelo:**
```
Cliente: "Preciso de manutenção no meu ar de 12.000 BTUs, Split Springer, 5 anos"
  ↓
Chatbot Refrimix:
  → Classifica: manutenção, 12k BTU, marca Springer, 5 anos = risco médio
  → Preço fixo: R$50 análise técnica (ou incluso se aprovar orçamento)
  → Pergunta: "Qual bairro?" → "Zona Sul, SP" → Técnico agendado
  → Envia alerta para admin com contexto completo
  → Cliente recebe: "Técnico confirmado para amanhã 14h. Histórico: Springer 12k 2020, últimas manutenções: nenhuma."
```

**Isso nenhuma empresa faz.**

## 5. GLOSSÁRIO HVAC-R EM PT-BR

| Termo | Definição |
|-------|-----------|
| **HVAC-R** | Heating, Ventilation, Air Conditioning and Refrigeration — Climatização e Refrigeração |
| **BTU** | British Thermal Unit — Unidade de medida de capacidade térmica (1 BTU ≈ 0,293 Wh) |
| **Split** | Sistema de ar-condicionado em duas unidades (evaporadora + condensadora) |
| **Inverter** | Tecnologia de variação de frequência do compressor para economia de energia (até 70%) |
| **Chiller** | Sistema de climatização centralizado para grandes edificações |
| **Compressor** | Unidade coração do sistema de refrigeração — comprime o fluido refrigerante |
| **Fluido Refrigerante** | Gás que absorve e libera calor no ciclo de refrigeração (R-410A, R-32, R-290) |
| **Serpentina** | Trocador de calor (evaporador ou condensador) |
| **COP** | Coefficient of Performance — Eficiência energética do sistema |
| **SEER** | Seasonal Energy Efficiency Ratio — Índice de eficiência energética sazonal |
| **ABRAVA** | Associação Brasileira de Refrigeração, Ar Condicionado, Ventilação e Aquecimento |
| **CLUBE DO FRIO** | Programa de capacitação e conteúdo para profissionais de refrigeração |
| **BTU/h** | BTU por hora — Capacidade de refrigeração |
| **Hi Wall** | Tipo de instalação de ar-condicionado de parede (evaporadora alta) |
| **Cassette** | Evaporadora de teto embutida (4 vias ou 1 via) |
| **Multi-Split** | Um condensadora alimentando vários evaporadoras |
| **Window** | Sistema de uma única unidade (monobloco) |
| **Minisplit** | Versão compacta de split para ambientes pequenos |
| **Free Cooling** | Uso de ar externo para climatização quando temperatura permite |
| **RTU** | Rooftop Unit — Sistema de climatização para telhados |
| **DX** | Direct Expansion — Expansão direta do fluido refrigerante |
| **AHU** | Air Handling Unit — Unidade de tratamento de ar |
| **Piping** | Sistema de tubulações de fluido refrigerante |
| **WHR** | Waste Heat Recovery — Recuperação de calor residual |
| **IAQ** | Indoor Air Quality — Qualidade do ar interior |
| **MOP** | Maximum Operating Pressure — Pressão máxima de operação |
| **ODS** | Ordem de Serviço — Documento de trabalho do técnico |
| **PMOC** | Plano de Manutenção, Operação e Controle — Obrigatório para sistemas >10 TR |
| **ART** | Anotação de Responsabilidade Técnica — Engenheiro responsável |
| **VRF** | Variable Refrigerant Flow — Sistema de volume de refrigerante variável |
| **VRV** | Variable Refrigerant Volume — Versão Daikin do VRF |
| **Piso-teto** | Evaporadora de média/alta capacidade instalada no piso ou teto |
| **Splitão** | Sistema split de alta capacidade (>60.000 BTUs) |
| **Cassete** | Evaporadora de teto tipo cassette |
| **Hi-Wall** | Evaporadora de parede tipo hi-wall |
| **Condensadora** | Unidade externa do sistema split |
| **Evaporadora** | Unidade interna do sistema split |
| **Carga de gás** | Quantidade de fluido refrigerante no sistema (kg) |
| **Vazamento** | Perda de fluido refrigerante por falha no sistema |
| **Checkout** | Termo usado para aferição de pressão/temperatura |

## 7. RECOMENDAÇÕES PARA O CHATBOT REFRIMIX

1. **Foco em emergências** — 40% das buscas ocorrem fora do horário comercial
2. **Qualificação de leads** — Questionário: tipo sistema, marca, idade, urgência, orçamento, localização
3. **Preço fixo antes da visita** — Evitar "agenda orçamento" genérico
4. **Integração com agenda** — Confirmar automaticamente slot disponível
5. **Contexto para técnico** — Técnico chega sabendo marca, modelo, histórico
6. **Manutenção preventiva** — Foco em retention, não apenas captação
7. **PT-BR absoluto** — Nenhum termo PT-PT, ES ou scripts não latinos
8. **Commercial router como autoridade** — LLM nunca decide preço
