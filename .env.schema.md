## REFRIMIX_CORE_VERSION

**Valores:** `v2` | `legacy`
**Default:** `v2`
**Descrição:** Qual pipeline usar como runtime principal.

- `v2` → `refrimix_core.domain.pipeline` (determinístico, sem LLM, recomendado)
- `legacy` → `agent_graph.graph.graph` (LangGraph, requer OPENAI_API_KEY)

**Importante:** Para o repo limpo, usar SEMPRE `v2`.

---

## DATABASE_URL

**Formato:** `postgresql://user:password@host:5432/database_name`
**Obrigatório:** Sim
**Descrição:** URL de conexão do PostgreSQL (banco `whatsapp_rag`).

**Não usar:** `100.66.232.72` hardcoded — usar nome DNS ou variável.

---

## REDIS_URL

**Formato:** `redis://host:port` ou `redis://user:password@host:port`
**Default:** `redis://localhost:6379`
**Descrição:** Connection pool do Redis para filas e cache.

---

## EVOLUTION_API_URL

**Formato:** URL completa (ex: `https://evolution.example.com`)
**Obrigatório:** Sim (para produção)
**Descrição:** Base URL da Evolution API.

---

## EVOLUTION_INSTANCE

**Formato:** string (nome da instância na Evolution API)
**Obrigatório:** Sim
**Descrição:** Nome da instância WhatsApp conectada.

---

## EVOLUTION_API_KEY

**Formato:** string (token da Evolution API)
**Obrigatório:** Sim
**Descrição:** API key para autenticação na Evolution API.

---

## OWNER_PHONE

**Formato:** número WhatsApp com código país (ex: `5511999999999`)
**Obrigatório:** Sim
**Descrição:** Telefone do dono para alertas e emergência.

---

## MINIMAL_MVP_ENABLED

**Valores:** `0` | `1`
**Default:** `1`
**Descrição:** Quando `1`, desabilita LangGraph do caminho crítico do worker.
O worker usa `process_mvp_message` do `app/mvp_attendance.py` em vez de `build_graph()`.

**Importante:** Manter `1` para o repo limpo (desabilita LangGraph legado).

---

## RAG_ENABLED

**Valores:** `0` | `1`
**Default:** `0`
**Descrição:** Habilita busca RAG no Qdrant para增强respostas com contexto.

**Custo:** Requer `QDRANT_URL` configurado + Minimax API key.

---

## TTS_ENABLED

**Valores:** `0` | `1`
**Default:** `0`
**Descrição:** Habilita síntese de voz (TTS) para resposta em áudio.

**Custo:** Requer `CHATTERBOX_TTS_URL` configurado.

---

## VISION_ENABLED

**Valores:** `0` | `1`
**Default:** `0`
**Descrição:** Habilita análise de imagem (OCR + classificação).

**Custo:** Requer Minimax API key.

---

## STT_ENABLED

**Valores:** `0` | `1`
**Default:** `0`
**Descrição:** Habilita transcrição de áudio para texto (Speech-to-Text).

---

## PTBR_SANITIZER_ENABLED

**Valores:** `0` | `1`
**Default:** `0`
**Descrição:** Habilita normalizador PT-BR (correct spelling, remove gringo).

---

## CHATTERBOX_TTS_URL

**Formato:** URL (ex: `http://localhost:5000`)
**Descrição:** URL do servidor Chatterbox TTS local.

---

## OMNIROUTE_ENABLED

**Valores:** `0` | `1`
**Default:** `0`
**Descrição:** Habilita roteamento omnichannel via OmniRoute.

---

## QDRANT_URL

**Formato:** URL (ex: `http://localhost:6333`)
**Descrição:** URL do Qdrant (só usado se `RAG_ENABLED=1`).

---

## QDRANT_COLLECTION

**Formato:** string
**Default:** `hermes_hvac_rag_service_staging`
**Descrição:** Nome da collection Qdrant para embeddings RAG.

---

## MINIMAX_API_KEY

**Formato:** string (API key do MiniMax)
**Descrição:** Chave da API MiniMax (usada por RAG e VISION se habilitados).

---

## LLM_API_BASE

**Formato:** URL
**Default:** `https://api.minimax.chat`
**Descrição:** Base URL da API do MiniMax.

---

## llm_model

**Valores:** `mini-max` | `MiniMax-Text-01`
**Default:** `MiniMax-Text-01`
**Descrição:** Modelo LLM para features que usam geração de texto.
